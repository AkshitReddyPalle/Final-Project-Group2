This Streamlit application provides an interactive interface to test **five deep-learning models** trained on the Human Action Recognition (HAR) dataset:

- **EfficientNet-B1** 
- **VGG16**
- **ResNet-50**
- **InceptionV3**
- **Custom CNN** 

The app allows you to upload any HAR image and either:
- Run **one selected model**, or  
- Run **ALL models together** to compare predictions side-by-side.

This makes it perfect for presentations, demo videos, evaluations, and showing model behavior in real time.
